Deskripsi Aplikasi

Aplikasi ini merupakan website portofolio pribadi berbasis Single Page Application (SPA) dengan ReactJS. Website berfungsi untuk menampilkan data diri mahasiswa dalam bentuk digital, meliputi profil, keahlian, dan pengalaman. Semua informasi ditampilkan secara ringkas dalam satu halaman sehingga mudah dibaca tanpa harus berpindah halaman.

Tampilan aplikasi dibuat sederhana dan responsif, menggunakan Tailwind CSS agar elemen antarmuka terlihat rapi dan konsisten. Data mahasiswa disimpan dalam berkas JSON sehingga memudahkan saat ingin memperbarui informasi di kemudian hari tanpa harus mengubah struktur komponen.

Cara Menjalankan Aplikasi

Pastikan Node.js sudah terpasang di perangkat.

Buka terminal atau Command Prompt kemudian masuk ke folder proyek React.

Instal semua dependensi terlebih dahulu dengan menjalankan perintah npm install.

Setelah proses instalasi selesai, jalankan aplikasi menggunakan perintah npm start.

Buka browser kemudian akses alamat http://localhost:3000 untuk melihat aplikasi berjalan.

Jika ingin melakukan deployment, jalankan perintah npm run build untuk menghasilkan folder build yang siap diunggah ke layanan hosting seperti Vercel atau Netlify.
